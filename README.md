Awesome pylab created by Gronemeyer

## Install it from PyPI

```bash
pip install pylab
```

## Usage

```py
from pylab import BaseClass
from pylab import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m pylab
#or
$ pylab
```
